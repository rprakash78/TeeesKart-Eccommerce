# TeeesKart-Eccommerce
Teeeskart is a Eccommerce Project which helps the user to virtually buy products.

Modules:

Admin Module:
  
  1. Manage Activities
  2. Manage Products
  3. Manage Users
  4. Manage Orders

User Module:
  1. Login or Register
  2. View Product
  3. Add to cart
  4. Buy Product
